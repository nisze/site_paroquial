O Sistema Digital Paroquial Nossa Senhora Aparecida será desenvolvido como projeto acadêmico conforme as especificações técnicas e requisitos funcionais definidos neste PRD.

Importante: O sistema não será hospedado ou colocado em produção neste momento

OBJETIVOS DO PROJETO
1. Objetivo Principal

DIGITALIZAÇÃO COMPLETA DA PASTORAL

Criar uma plataforma digital integrada que facilite o acesso dos fiéis aos recursos espirituais, informações paroquiais e conteúdo litúrgico, promovendo uma evangelização eficaz na era digital.

2. Objetivos Específicos
  • Implementar sistema de consulta à Bíblia Sagrada com busca avançada
  • Integrar liturgia diária oficial da Igreja
  • Desenvolver banco de dados completo dos Santos Católicos
  • Criar sistema de informações paroquiais e horários de missa
  • Desenvolver interface responsiva para todos os dispositivos
  • Garantir conformidade legal total com direitos autorais
  • Construir backend robusto com API REST completa
  • Produzir documentação técnica e legal abrangente

FUNCIONALIDADES A SEREM IMPLEMENTADAS
1. Sistema Bíblico Completo
  • Consulta por livro, capítulo e versículo
  • Busca por palavra-chave em todo o texto
  • Navegação intuitiva entre livros
  • Interface responsiva para todos os dispositivos
  • Integração com API oficial bible-api.com
  • Tradução João Ferreira de Almeida (domínio público)
2. Liturgia Diária Oficial
  • Integração com API oficial da Igreja
  • Primeira leitura, salmo responsorial, evangelho
  • Atualização automática diária
  • Layout otimizado para leitura
  • Compatível com liturgia romana oficial
3. Santos Católicos
  • Banco de dados com 365+ santos
  • Santo do dia automático
  • Busca por nome e data de festa
  • Biografias completas e oficiais
  • Imagens legais (domínio público)
  • Sistema de validação de fontes
4. Sistema de Informações Permanentes

Funcionalidade Crítica

Sistema para gerenciar informações permanentes como horários de missa, catequese e grupos paroquiais

  • 15 Tipos de Informações: MISSA, CATEQUESE, GRUPO_ORACAO, PASTORAL_JUVENTUDE, etc.
  • CRUD Completo: Criar, editar, listar e excluir informações
  • API REST: Endpoints para administração e consulta pública
  • Filtros Avançados: Por tipo, comunidade e status ativo
  • Área Administrativa: Interface dedicada para o pároco
  • Facilita Manutenção: Sem necessidade de alterar código para updates
5. Informações Paroquiais
  • 7 comunidades paroquiais detalhadas
  • Horários de missa atualizados
  • Contatos de cada comunidade
  • Endereços e mapas de localização
  • Informações sobre pároco e equipe

  • Controle de permissões para mensagens
  • Repository com queries especializadas

CONCLUSÃO

PROJETO ACADÊMICO PARA APROVAÇÃO

O Sistema Digital Paroquial Nossa Senhora Aparecida será desenvolvido como projeto acadêmico com excelência técnica e aplicação prática dos conhecimentos adquiridos no curso de Análise e Desenvolvimento de Sistemas.


Escopo do Projeto Acadêmico:

  • Plataforma Web Completa - Angular 20 + Spring Boot 3.5.5
  • Banco de Dados - H2 Database (ambiente de desenvolvimento)
  • Funcionalidades Principais - Bíblia, Liturgia, Santos, Comunidades
  • APIs Integradas - Bible API e Liturgia Diária Oficial
  • Documentação Técnica - PRD, diagramas, código documentado

Cronograma Acadêmico:

  • Duração: 7 semanas (desenvolvimento backend e frontend)
  • 4 fases de desenvolvimento técnico
  • Validações acadêmicas com orientadora
  • Entrega: Sistema funcional em ambiente local

Importante: Este é um projeto exclusivamente acadêmico. O sistema não será hospedado em servidor nem colocado em produção. O Pe. Luiz Paulo Batista Braga aprova o desenvolvimento para fins de aprendizado e avaliação acadêmica, não para uso efetivo pela paróquia neste momento.
